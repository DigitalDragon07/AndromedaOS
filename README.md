This is a project to create an OS inside the IDE TigerJython. It uses Jython, which is a combination of Python 2.7 synatx and JVM.
It as such doesn't really work like an OS, since it can't run on it's own. It is nevertheless useful for making apps and other things that require some sort of UI as this is already implemented.
Maybe I should also mention that it isn't a all to serious project. Despite that, I still put in a lot of effort so please check it out!
